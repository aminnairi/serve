# @aminnairi/serve

```console
$ npx @aminnairi/serve --folder client
Serving files from the client folder at http://127.0.0.1:8000. Hit CTRL+C at any time to stop.
```

## Requirements

- NPM
- Node

## Usage

### NPX

```console
$ npx @aminnairi/serve --help
```

### NPM

```console
$ npm install @aminnairi/serve
$ node ./node_modules/.bin/serve --help
```

### Global

```console
$ sudo npm install --global @aminnairi/serve
$ serve --help
```

## Examples

### Folder

```console
$ serve --folder public
```

### Host

```console
$ serve --host 0.0.0.0
```

### Port

```console
$ serve --port 5000
```

### Single-page application

```console
$ serve --spa
```

### Verbose

```console
$ serve --verbose
```

### Help

```console
$ serve --help
```
